Welcome to the source code of my personal portfolio. This project abandons standard graphical interfaces in favor of a **command-line inspired experience**, reflecting my background in **Computer Engineering**, **Linux Systems**, and **Backend Logic**.

Instead of navigating a traditional menu, visitors interact with a simulated shell environment‚Äîexecuting commands to fetch projects, skills, and contact protocols.

## üöÄ Key Features

* **Interactive CLI:** A JavaScript-driven shell that parses user commands (e.g., `./contact.sh`, `cat about.txt`).
* **Hybrid Navigation:** Includes clickable "menu commands" for non-technical visitors (GUI fallback).
* **Dynamic ASCII Art:** Features custom-rendered ASCII graphics (Tux) that adapt to screen size.
* **System-Log Aesthetics:** Redirects and loading screens mimic server boot sequences (`STDOUT` style logging).
* **Optimized Performance:** Built with vanilla JS and CSS to ensure minimal latency.
